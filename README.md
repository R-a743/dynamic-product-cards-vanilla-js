# Dinamik Ürün Kartları

Bu proje, **JavaScript kullanarak dinamik ürün kartları oluşturma** ve bunları filtreleme, sıralama veya arama işlemleri yapabilme özelliklerini gösterir. Kullanıcılar, ekrandaki ürün kartlarını belirli kriterlere göre görüntüleyebilir.

---
♦ CANLI DEMO :  https://r-a743.github.io/dynamic-product-cards-vanilla-js/
 
## 🔹 Özellikler

1. **Dinamik Ürün Kartları Oluşturma**
   - Kullanıcı, kaç ürün gösterileceğini belirleyebilir.
   - Fiyatına göre pahalı ürünler özel renkle vurgulanır.

2. **Fiyata Göre Sıralama**
   - Ürünleri küçükten büyüğe doğru sıralayabilirsiniz.

3. **Pahalı Ürünleri Gösterme**
   - Fiyatı 10.000₺ üstü olan ürünler listelenir.

4. **Ürün Arama**
   - Arama kutusuna ürün adı girerek ürünleri filtreleyebilirsiniz.
   - Arama, büyük/küçük harf duyarsızdır.

5. **Responsive Tasarım**
   - Ürün kartları farklı ekran boyutlarına uygun şekilde dizilir.

---

## 🔹 Kullanılan Teknolojiler

- **HTML5**: Sayfa yapısı ve temel elementler.
- **CSS3**: Kart tasarımı, renkler, gölge ve responsive düzen.
- **JavaScript**: Dinamik veri oluşturma, filtreleme, sıralama ve arama işlemleri.

---

## 🔹 Kullanım

1. **Projeyi açın**
   - `index.html` dosyasını tarayıcıda açın.

2. **Kaç ürün göstermek istediğinizi belirleyin**
   - "Kaç ürün gösterilsin?" kutusuna sayı girin ve **Oluştur** butonuna tıklayın.

3. **Fiyata göre sıralama**
   - **Fiyata Göre Sırala** butonuna tıklayın.

4. **Pahalı ürünleri gösterme**
   - **Pahalı Ürünler** butonuna tıklayın (fiyatı 10.000₺ üzeri olanlar).

5. **Ürün arama**
   - Arama kutusuna ürün adını yazın; liste otomatik filtrelenecektir.
